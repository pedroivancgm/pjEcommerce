Se você quer testar o aplicativo, no vscode você precisará de dois terminais.<br>
Primeiro terminal: <br>
cd frontend<br>
npm run dev<br>
acessar a url local para vizualizar o frontend<br>
<br>
Segundo terminal:<br>
cd backend<br>
rails s<br>
<br>
